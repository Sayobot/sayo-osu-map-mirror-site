小夜是什么？

小夜是一个基于酷Q、osu!api open beta 的QQ bot.

主要编写语言是C++，辅以脚本语言Ruby、powershell、cmd（批处理）.

小夜目前能做的事情还不是很多，但可以帮你查一下osu游戏的基本数据，然后生成一个资料卡。
